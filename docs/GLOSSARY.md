## Glossary

| Термин | Значение |
|---|---|
| **Paytable** | Таблица выплат |
| **Full Pay** | Лучшая (для игрока) версия paytable |
| **Short Pay** | Урезанная версия с меньшим RTP |
| **9/6** | 9 за Full House, 6 за Flush (за 1 монету) |
| **RTP** | Return To Player — % возврата при оптимальной игре |
| **House Edge** | 100% - RTP |
| **Wild card** | Заменяет любую другую |
| **Kicker** | Пятая карта при 4oaK, влияет в DDB |
| **Pat hand** | Готовая комбинация без замены |
| **Max Bet** | 5 монет, активирует бонус Royal Flush |
| **Deal** | Начальная раздача |
| **Draw** | Замена незафиксированных |
| **Hold** | Фиксация карты |
| **Credits** | Виртуальная валюта |
| **Denomination** | Номинал монеты |
| **Natural** | Без wild-карт |
| **Variance / Volatility** | Колебания баланса |

## Историческая справка

- Первые машины — середина 1970-х
- 1979 — SIRCOMA (будущая IGT) выпустила Draw Poker
- 1980-е — стал одной из самых популярных казино-игр
- IGT Game King — референсная машина для визуального стиля
